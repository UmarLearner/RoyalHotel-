# 🏨 Royal Stay Hotel Website

A fully responsive luxury hotel website built using **HTML, CSS (Flexbox), and JavaScript**.  
This project is designed as a modern hotel landing page with smooth UI, animations, and mobile responsiveness.

## ✨ Features

- 🏨 Luxury hotel UI design
- 📱 Fully responsive (Mobile, Tablet, Desktop)
- 🎨 Blue resort modern theme

- ⚡ Smooth scrolling navigation
